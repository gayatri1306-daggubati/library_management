BASIC FORMULAE

1. Network Performance Metrics

Bandwidth = Data Transferred / Time Taken
Latency = Propagation Delay + Transmission Delay + Queuing Delay + Processing Delay
Propagation Delay = Distance / Propagation Speed
Transmission Delay = Packet Size (bits) / Bandwidth (bps)
