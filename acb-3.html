BASIC FORMULAE

1. Basic Signal Properties

ConceptFormula
Even Signalx_even(t) = (x(t) + x(-t)) / 2
Odd Signalx_odd(t) = (x(t) - x(-t)) / 2
EnergyE = ∫ |x(t)|² dt
PowerP = lim(T→∞) (1/2T) ∫ |x(t)|² dt

2. Fourier Series and Transform

3. Laplace Transform

ConceptFormula
Laplace TransformX(s) = ∫ x(t) e^(-st) dt
Inverse Laplacex(t) = (1/2πj) ∫ X(s) e^(st) ds

4. Z-Transform

ConceptFormula
Z-TransformX(z) = Σ x[n] z^(-n)

5. Convolution

ConceptFormula
Continuous-Time Convolutiony(t) = ∫ x(τ) h(t - τ) dτ
Discrete-Time Convolutiony[n] = Σ x[k] h[n - k]
