BASIC FORMULAE

1. Differential Calculus

f'(x) = limh→0 (f(x+h) - f(x)) / h
(uv)' = u'v + uv'
(u/v)' = (u'v - uv') / v²

2. Integral Calculus

∫ xⁿ dx = (xⁿ⁺¹)/(n+1) + C
∫ e^x dx = e^x + C

3. Matrices and Determinants

|A| = ad - bc (for 2×2 matrix)
A⁻¹ = (1/|A|) Adj(A)

4. Differential Equations

dy/dx + P(x)y = Q(x)

5. Laplace Transforms

L{1} = 1/s
L{e^at} = 1/(s-a)

6. Fourier Series

f(x) = a₀ + Σ (aₙ cos(nπx/L) + bₙ sin(nπx/L))

7. Vector Calculus

∇f = (∂f/∂x, ∂f/∂y, ∂f/∂z)

8. Probability and Statistics

E(X) = Σ x P(x)
σ² = E(X²) - [E(X)]²
